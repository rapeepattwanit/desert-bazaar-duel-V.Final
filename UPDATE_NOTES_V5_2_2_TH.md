# อัปเดต v5.2.2 — Mobile Image/Touch Performance Patch

แก้เฉพาะปัญหามือถือที่ภาพการ์ดโหลดช้า/ไม่ขึ้น และมีอาการกระตุกเวลาแตะเล่น โดยไม่เปลี่ยน logic เกม

## สิ่งที่แก้

1. เปลี่ยน path ภาพการ์ดหลักจาก `.png` เป็น `.webp`
   - ลดขนาดภาพจากประมาณ 2.8–3.1MB ต่อใบ เหลือประมาณ 80–102KB ต่อใบ
   - มือถือโหลดและ decode ภาพเบาลงมาก

2. เพิ่ม fallback กลับไปใช้ `.png`
   - ถ้า browser ใดเปิด `.webp` ไม่ได้ ระบบจะลองโหลด `.png` แทน

3. เปลี่ยนการโหลดรูปการ์ดจาก `lazy` เป็น `eager` พร้อม `decoding="async"`
   - การ์ดในเกมเป็นรูปที่ต้องเห็นทันที จึงไม่ควรรอ lazy-load บนมือถือ

4. preload รูปการ์ดล่วงหน้า
   - ช่วยให้เมื่อเข้าเกมแล้วรูปขึ้นเร็วขึ้น และลดอาการกระตุกตอนแตะครั้งแรก

5. หน่วงเพลงพื้นหลังบนอุปกรณ์ touch จนกว่ารูปการ์ดจะพร้อม
   - ป้องกันไฟล์เพลงขนาดใหญ่แย่ง bandwidth/CPU กับรูปในช่วงแรก

6. ลด effect CSS หนักบนมือถือ
   - ปิด backdrop blur และ filter/transform ที่หนักกับ GPU มือถือเฉพาะ touch device
   - คง layout และ gameplay เดิมไว้

7. เพิ่ม cache header สำหรับภาพการ์ดและเสียง effect
   - ลดการโหลดซ้ำระหว่างเล่นหรือ reconnect

## หมายเหตุ

- Laptop/desktop layout เดิมยังคงอยู่
- ไม่แตะ rule เกม, WebSocket protocol, scoring, reconnect หรือระบบห้อง
