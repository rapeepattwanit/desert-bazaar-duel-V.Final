# UPDATE v4.1

ปรับปรุงตามรายการล่าสุด:

1. คู่มือหัวข้อคะแนนและโบนัส เพิ่มรายละเอียดโบนัสขายหลายใบ:
   - ขาย 3 ใบ ได้โบนัส 1–3 แต้ม
   - ขาย 4 ใบ ได้โบนัส 4–6 แต้ม
   - ขาย 5 ใบขึ้นไป เช่น 5, 6 หรือ 7 ใบ ได้โบนัส 8–10 แต้ม
2. เพลงพื้นหลังจะเริ่มเล่นเฉพาะหลังเข้าไปในห้องเกมแล้วเท่านั้น ไม่เล่นบนหน้า Home
3. เพิ่มปุ่ม Setting ในหน้า Home
4. เปลี่ยนคำโปรยหน้า Home เป็น “สุดยอดบอร์ดเกมค้าขาย Online ที่มียอดดาวโหลดมากกว่า 1ล้านชุด”
5. ใน Layout “คำสั่ง” ย้ายคำสั่ง “แลก” ขึ้นมาเป็นรายการแรก แล้วตามด้วยรายการขายสินค้า
6. เหรียญสินค้าจะเปลี่ยนสีตามราคาปัจจุบัน:
   - ราคายังสูง ใช้สีขาว/สว่าง
   - ราคาเริ่มลด ใช้สีเหลือง
   - ราคาต่ำสุด ใช้สีแดง
7. ตัวเลขราคาเหรียญสินค้าเด่นขึ้น เห็นง่ายขึ้น
8. เพิ่มเสียง Action พร้อมโฟลเดอร์ `assets/audio/effects/` สำหรับเปลี่ยนเสียงเอง:
   - `notification-chime.wav`
   - `sell-money.wav`
   - `camel.wav`
   - `pick-card.wav`
   - `trade.wav`
   - `winner.wav`
9. ปรับหมวดเสียงใน Setting เป็น 3 หมวดชัดเจน:
   - Master sound 0–5 พร้อมเปิด/ปิด
   - เสียง effect ทั้งหมด 0–5 พร้อมเปิด/ปิด
   - เสียงเพลงพื้นหลัง 0–5 พร้อมเปิด/ปิด

## การเพิ่มเพลงพื้นหลังเอง

ใส่ไฟล์ไว้ที่:

```txt
assets/audio/
```

แล้วเพิ่ม path ใน `app.js` ตรงตัวแปร `BACKGROUND_TRACKS` เช่น:

```js
const BACKGROUND_TRACKS = [
  '/assets/audio/casino-jazz-loop.wav',
  '/assets/audio/my-jazz-1.mp3',
  '/assets/audio/my-jazz-2.mp3'
];
```

## การเปลี่ยนเสียง Effect เอง

ใส่ไฟล์เสียงไว้ที่:

```txt
assets/audio/effects/
```

ถ้าใช้ชื่อไฟล์เดิม เช่น `sell-money.wav` ไม่ต้องแก้โค้ดเพิ่ม
