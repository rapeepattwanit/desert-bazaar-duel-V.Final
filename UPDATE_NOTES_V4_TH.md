# Desert Bazaar Duel V4 Update

สิ่งที่ปรับเพิ่ม:

1. ตลาดกลางตอนเริ่มรอบสุ่มจากสำรับจริงทุกครั้ง ไม่เริ่มด้วยอูฐ 3 ใบตายตัวอีกแล้ว
2. เพิ่มระบบเติม/เช็กตลาดกลางเพื่อกันบัคตลาดเหลือ 2 ใบแล้วเล่นต่อไม่ได้
3. เปลี่ยนเสียงแจ้งเตือนเป็น notification chime
4. เพิ่มปุ่ม Setting พร้อมตัวเลือกเสียงแจ้งเตือน, เสียงเกม, popup แจ้งเตือน, คัดลอกลิงก์ และ Home
5. เพิ่ม popup ตอนจบเกม พร้อมคะแนน 2-0 / 2-1, เสียงผู้ชนะ, ปุ่มเริ่มใหม่ และสร้างห้องใหม่
6. เพิ่มระบบเพลงพื้นหลังจากโฟลเดอร์ assets/audio โดยสุ่มไฟล์และสุ่มจุดเริ่มเล่น
7. ช่องจำนวนขายตั้งค่าเริ่มต้นตามจำนวนการ์ดที่มี เช่น มีเพชร 3 ใบ จะตั้งขาย 3 ใบทันที
8. ปรับ Layout ให้ “มือของคุณ” อยู่ตรงกลางระหว่าง “ตลาดกลาง” และ “คำสั่ง”
9. การ์ดในสนามและมือผู้เล่นแสดงเฉพาะรูป PNG ไม่ใส่ text ทับบนรูป
10. server.js ใช้ path รูปการ์ดเป็น .png แล้ว

การ deploy:

- อัปโหลดไฟล์ทั้งหมดขึ้น GitHub ยกเว้น node_modules
- ใช้ Build Command: npm install --no-audit --no-fund
- ใช้ Start Command: npm start
- ตั้ง NODE_VERSION = 22.x
