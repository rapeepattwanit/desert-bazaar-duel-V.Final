# Update Notes v5.1 Bugfix

แก้บัคสำคัญจาก v5:

1. แก้บัค Join ห้องแล้วขึ้น “ห้องเต็มแล้ว” ทั้งที่ยังมีที่ว่าง
   - สาเหตุ: หลังสร้างห้อง array `players` มีความยาวแค่ 1 ทำให้ระบบหา slot ผู้เล่นที่ 2 ไม่เจอ
   - แก้โดยกำหนด slot ผู้เล่นเป็น `[null, null]` และตรวจ 2 ที่นั่งแบบ explicit

2. ปรับ reconnect / reclaim slot
   - ถ้าผู้เล่นเดิม reconnect ด้วย `clientId` เดิม จะกลับเข้า slot เดิม
   - ถ้าห้องไม่มีคนออนไลน์เลย ผู้เล่นใหม่สามารถ reclaim slot ได้ทันที
   - ถ้าห้องเต็มจริง จะแจ้งว่า “มีผู้เล่นออนไลน์ครบ 2 คนแล้ว”

3. แก้รูปการ์ดโหลดไม่ขึ้นบางใบ
   - เปลี่ยน path เริ่มต้นใน `server.js` จาก `.webp` เป็น `.png` เพื่อให้ตรงกับไฟล์การ์ดหลัก
   - เพิ่ม fallback ฝั่ง client: ถ้ารูปที่ส่งมาพัง จะลองโหลด `/assets/cards/<card>.png` อัตโนมัติ

ไฟล์ที่แก้หลัก ๆ:
- `server.js`
- `public/app.js`
