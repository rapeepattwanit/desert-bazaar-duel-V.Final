# Desert Bazaar Duel v5.2.0 Bugfix

แก้ตามรายการล่าสุด:

1. ปรับระบบความดังเสียงให้ลด/เพิ่มได้จริงขึ้นด้วย volume curve และ Web Audio สำหรับเสียง effect
2. ลดอาการภาพการ์ดกระตุกบนมือถือ โดยไม่ rebuild DOM/โหลดรูปใหม่ทุกครั้งที่เลือกการ์ด
3. เอาการกดยืนยัน “หยิบสินค้า 1 ใบ” ออก แตะการ์ดสินค้าในตลาดแล้วหยิบทันที
4. ปรับ layout บน laptop/desktop ให้ “มือของคุณ” อยู่ใต้ “ตลาดกลาง” และ “คำสั่ง” อยู่ด้านข้าง
5. ช่องกรอกรหัสห้องบนหน้า Home บังคับเป็นตัวพิมพ์ใหญ่และกรองเฉพาะ A-Z / 0-9
6. ข้อความ popup ผู้ชนะเปลี่ยนเป็น “ยินดีด้วยคุณคือผู้ชนะ” โดยไม่มีอีโมจิในข้อความ

หมายเหตุ: ไฟล์ .gitkeep ไม่เกี่ยวกับการโหลดเสียงหรือรูปภาพ สามารถมีหรือไม่มีได้เมื่อมีไฟล์จริงในโฟลเดอร์แล้ว
