# Effect Audio

ใส่ไฟล์เสียง effect เป็นตัวพิมพ์เล็กทั้งหมดเพื่อให้ใช้ได้บน Render/Linux:

```txt
notification-chime.mp3  = แจ้งเตือนถึงตาคุณ
pick-card.mp3           = คลิก/เลือกการ์ด และหยิบสินค้า 1 ใบ
sell-money.mp3          = ขายสินค้า/ได้เงิน
camel.mp3               = เก็บอูฐทั้งหมด
trade.mp3               = แลกเปลี่ยนสำเร็จ
winner.mp3              = ชนะรอบ/ชนะเกม
```

ถ้าจะเปลี่ยนเสียง ให้ใช้ชื่อไฟล์เดิมและนามสกุล `.mp3` จะไม่ต้องแก้โค้ด
